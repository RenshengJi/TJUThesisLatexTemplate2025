# TJUThesisLatexTemplate2024

Latex template for TJU thesis. Modified from [here](https://code.google.com/p/tjuthesis)

# 说明

天津大学本科生毕业论文 LaTeX 模版。

在前人版本的基础上按照 2024 届本科生毕业论文要求进行部分修改。

在 Overleaf 中使用 xeletex 编译方式
